BF Interpreter

psurge1.github.io/bf_interpreter hosts a BF interpreter built with Python, PyScript, JavaScript, HTML and CSS, written by psurge1.


BF code can be typed into the blue, resizable input box. BF code typed in the input box is executed when the run button is pressed and the output is displayed to the orange output box.


Symbol Description
+ increments the value in the active cell
- decrements the value in the active cell
, inputs a character to save its ascii numerical representation in the active cell
. outputs the ascii character representation of the number in the active cell
> increments the data pointer - moves to the cell on the right
< decrements the daata pointer - moves to the cell on the left
